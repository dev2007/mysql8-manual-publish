# 2.9.3 源码安装的 MySQL 布局

默认情况下，当你在从源代码编译 MySQL 之后安装它时，安装步骤会在 /usr/local/mysql 下安装文件。安装目录下的组件位置与二进制发行版相同。参阅表2.3，“通用 Unix/Linux 二进制包的MySQL安装布局”和章节 2.3.1，“Microsoft Windows 上的 MySQL 的安装布局”。要配置不同于默认设置的安装位置，请使用章节 2.9.7，“MySQL 源配置选项”中描述的选项。
